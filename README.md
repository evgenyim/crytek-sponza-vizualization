# crytek-sponza-vizualization
OpenGL visualization of crytek sponza scene

*List of features:*
- Sponza scene
- Moving model
- Textures
- Ambient, diffuse, specular lighting
- Normal mapping
- Global + local light sources
- Shadows
- Reflections


## Build and Run
1. ``mkdir build && cd build``
2. `cmake ..`
3. `cmake --build .`
4. `./sponza_scene`
